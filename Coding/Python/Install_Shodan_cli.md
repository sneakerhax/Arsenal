# Install Shodan cli

The Shodan cli is tool written in Python that interacts with the Shodan API to get information

## Installing Shodan cli (Python3)

```
python3 -m pip install shodan
```

## Initialize with API key

```
shodan init <api_key>
```

## Test basic command

```
shodan host <ip_address>
```
